EL BAMBINO DANIEL — BACKEND FINAL CORREGIDO

ARCHIVOS QUE DEBE SUBIR A GITHUB
- Backend/server.js
- Backend/package.json

PASOS
1. Abra la carpeta Backend de su repositorio.
2. Use Add file > Upload files.
3. Suba server.js y package.json.
4. Acepte reemplazar los archivos existentes.
5. Presione Commit changes.
6. Render iniciará automáticamente un nuevo despliegue.

NO SUBA TOKENS A GITHUB.

ADMIN_PASSWORD o ADMIN_PIN
JWT_SECRET
ALLOWED_ORIGINS o ALLOWED_ORIGIN

ADMIN_WHATSAPP
