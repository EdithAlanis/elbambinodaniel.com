EL BAMBINO DANIEL — CORREO AUTOMÁTICO DE PEDIDOS

ESTE ZIP SOLO ACTUALIZA EL BACKEND.

ARCHIVOS A SUBIR
1. Abra GitHub.
2. Entre a la carpeta Backend.
3. Use Add file > Upload files.
4. Suba server.js y package.json.
5. Reemplace los existentes.
6. Presione Commit changes.
7. Espere a que Render termine el despliegue.

CONFIGURACIÓN EN RENDER
En el servicio elbambinodaniel-pedidos, abra Ambiente y agregue:

EMAIL_USER
La cuenta Gmail desde la que se enviarán los avisos.
Ejemplo: prosecogdl@gmail.com

EMAIL_APP_PASSWORD
Una contraseña de aplicación de Google de 16 caracteres.
NO use la contraseña normal del correo.

EMAIL_TO
prosecogdl@gmail.com

Después guarde los cambios. Render hará otro despliegue.

FUNCIONAMIENTO
Cuando el cliente envíe un pedido:
- El pedido se guarda en PostgreSQL.
- Se genera el folio.
- El cliente ve la confirmación.
- Se envía automáticamente un correo con cliente, teléfono, domicilio,
  productos, cantidades, total y observaciones.

IMPORTANTE
El pedido se guarda aunque el correo falle. En Registros de Render aparecerá
el motivo del error para poder corregirlo.
